How it works

Most online balancers solve a linear system and stop at the first solution. This one uses the Contejean–Devie algorithm to compute the Hilbert basis — the set of all minimal, irreducible integer vectors that satisfy atom conservation.

For simple equations the result is a single balanced form. For complex systems (like the NH₄ClO₄ example above) the algorithm enumerates every chemically distinct balanced pathway — up to 17 independent solutions.

Built in Rust, compiled to WebAssembly. All computation runs locally in your browser. No server, no tracking.
